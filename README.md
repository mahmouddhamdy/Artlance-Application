## Artlance

Artlance is a mobile application, a centralized platform that enables freelancers to build their own portfolio and showcase their work, and on the other hand ease the search task for clients and make the connection between them easier. Creating accounts for both user types, allows exploring accounts and request tasks for freelancers, and allows communication between clients and freelancers, besides searching either by category, name, or image.
